# QGen — Document Processing & AI Training Data Platform

QGen transforms any document into high-quality synthetic Q&A datasets for AI model training. Upload PDFs, Word docs, and images — get structured training data in minutes.

## Features

- Multi-format document support (PDF, Word, Excel, PowerPoint, Images)
- AI-powered schema generation and Q&A extraction
- Batch processing up to 100 files simultaneously
- Export in JSON, CSV, XML formats
- Comprehensive quality scoring across 4 dimensions
- Enterprise security and on-premise deployment options

## Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Install & Run

```bash
git clone https://github.com/arora-raghav/qgen.git
cd qgen
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

### Build

```bash
npm run build
```
